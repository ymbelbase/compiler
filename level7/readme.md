# Level 007 - Calculator with JIT

## Build
```bash
make
```

## Run
```bash
./calc
```

Then type expressions (end with `;`) and press Enter. Press Ctrl+D to execute.

## Examples
```
2+3;
5 * 10;
x = 5;
x + 3;
```

### ctrl+d (linux) or ctrl+z (windows)........for mac probably ctrl+d

## Clean
```bash
make clean
```
